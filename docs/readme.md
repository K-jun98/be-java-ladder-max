### `구현 목록`
- [X] 참여할 인원을 입력받는다.
- [X] 사다리의 높이를 입력받는다.
- [X] 입력받은 인원과, 높이로 2차 배열을 만든다.
  - [X] 사다리 라인을 표시한다
- [X] 랜덤 값을 생성한다.
  - [X] 랜덤 값의 따라 "-" or " " 를 입력한다.
- [X] 사다리의 상태를 화면에 출력한다.

### `2단계 - 리팩토링`
- [X] 메서드의 크기가 10라인을 넘지 않도록 구현한다.
- [X] 메서드의 들여쓰기를 1단계로 줄여라
- [X] else를 사용하지 않는다.
- [X] naming convention을 지키며 프로그래밍한다.
